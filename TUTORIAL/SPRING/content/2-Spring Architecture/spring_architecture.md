				<h3 class="subheading">Architecture :</h3>
				<p>Spring is well-organized architecture consisting  of seven modules. Modules in the Spring framework are:</p>
				<h3 class="subheading padding-left">Spring AOP :</h4>
				<p class="padding-left">One of the key components of Spring is the AOP framework. AOP is used in Spring:</p>
				<p class="padding-left">1.To provide declarative enterprise services, especially as a replacement for EJB declarative services. The most important such service is declarative transaction management, which builds on Spring's transaction abstraction.</p>
				<p class="padding-left">2.Allow users to implement custom aspects, complementing their use of OOP with AOP.</p>
				<h3 class="subheading padding-left">Spring ORM :</h4>
				<p class="padding-left">The ORM package is related to the database access. It provides integration layers for popular object-relational mapping APIs, including JDO, Hibernate and iBatis.</p>
				<h3 class="subheading padding-left">Spring WEB :</h4>
				<p class="padding-left">The Spring Web module is part of Spring's web application development stack, which includes Spring MVC.</p>
				<h3 class="subheading padding-left">Spring DAO :</h4>
				<p class="padding-left">The DAO (Data Access Object) support in Spring is primarily for standardizing the data access work using the technologies like JDBC, Hibernate or JDO.</p>
				<h3 class="subheading padding-left">Spring Context :</h4>
				<p class="padding-left">This package builds on the beans package to add support for message sources and for the Observer design pattern, and the ability for application objects to obtain resources using a consistent API.</p>
				<h3 class="subheading padding-left">Spring WEB MVC :</h4>
				<p class="padding-left">This is the Module which provides the MVC implementations for the web applications.</p>
				<h3 class="subheading padding-left">Spring Core :</h4>
				<p class="padding-left">This component provides the Dependency Injection features. The BeanFactory  provides a factory pattern which separates the dependencies like initialization, creation and access of the objects from your actual program logic.</p>
				<p class="padding-left">The Core package is the most import component of the Spring Framework.</p>
