<h4>What is Anchor Tag? what is use of Anchor Tag?</h4>

<p>The HTML Anchor Element (&lt;a&gt;) defines a hyperlink to a resource (another web page, a file, etc.) on the same page or any other page on the Web. it use <b>href</b> attribute indicates the link's destination</p>

<p>By default, links will appear as follows in all browsers:</p>
<ul>
	<li>An unvisited link is underlined and blue</li>
	<li>A visited link is underlined and purple</li>
	<li>An active link is underlined and red</li>
</ul>
